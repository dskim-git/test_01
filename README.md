스네이크 게임 (HTML/CSS/JS 단일 파일)

키보드 화살표 키(↑ ↓ ← →) 로 조작하는 스네이크 게임입니다.
별도의 빌드/설치 없이 하나의 HTML 파일만으로 실행됩니다.
래핑 모드(벽을 통과해 반대편으로 등장)와 난이도(속도) 설정, 최고 점수 저장 기능을 지원합니다.

!screenshot
(이미지가 없다면 docs/screenshot.png 경로에 스크린샷을 추가하거나 이 문구를 삭제하세요.)

✨ 주요 기능

단일 파일 실행: index.html 하나만으로 동작
조작: 화살표 키(↑ ↓ ← →), Space로 일시정지/재개
난이도(속도) 선택: 느림 / 보통 / 빠름
래핑 모드: 벽 충돌 시 반대편으로 이동 (옵션)
점수 & 최고 점수: localStorage를 활용한 브라우저 로컬 저장
반응형 UI: 데스크톱/노트북 화면에서 보기 편한 레이아웃


🧩 데모

로컬에서 index.html 파일을 브라우저로 열면 즉시 실행됩니다.
깃허브 Pages를 사용하려면 이 저장소의 Settings → Pages에서 브랜치/폴더를 지정하세요.


🚀 시작하기
요구 사항

최신 브라우저(Chrome, Edge, Firefox, Safari)
서버 불필요: 파일 더블클릭만으로 실행 가능

실행 방법

이 저장소를 클론 또는 다운로드
루트 디렉터리의 index.html 더블클릭 → 브라우저에서 실행
상단의 [시작] 버튼 클릭 후 화살표 키로 조작


🎮 조작 방법





























입력기능↑ / ↓ / ← / →이동Space일시정지/재개시작 버튼게임 시작일시정지 버튼일시정지/재개리셋 버튼현재 게임 상태 초기화

주의: 뱀은 180도 회전(정반대 방향 전환) 을 즉시 할 수 없습니다. (자기 몸에 바로 부딪히는 입력을 방지)


⚙️ 옵션 / UI 설명

속도(난이도): 셀렉트 박스에서 느림 / 보통 / 빠름 선택
래핑: 체크 시 보드의 경계에서 반대편으로 이동 (해제 시 벽 충돌은 게임오버)
점수/최고: 상단 배지로 표시, 최고 점수는 브라우저의 localStorage에 저장

🛠 커스터마이즈
아래 값들은 index.html 내부에서 바로 수정할 수 있습니다.

보드 크기:
JavaScriptconst COLS = 21; // 열 개수const ROWS = 21; // 행 개수// <canvas width="420" height="420">와 함께 조정 권장 (타일 크기 = width / COLS)더 많은 선 표시

속도 기본값/옵션:
<select id="speedSelect"> 옵션 값 및 selected 변경
컬러 테마:
<style>의 :root CSS 변수(예: --snake, --food, --bg) 변경
룰 확장:
step() 함수에 장애물, 포털, 특수 아이템(가속/감속/보너스 점수) 로직 추가


🧪 테스트 팁

키 입력이 반응하지 않으면 캔버스를 한 번 클릭해 포커스를 주세요.
브라우저 탭이 백그라운드로 가면 애니메이션이 느려질 수 있습니다(표준 동작).
최고 점수를 초기화하려면 개발자 도구에서 localStorage.removeItem('snake_best') 실행 또는 리셋 로직을 수정하세요.


🔒 개인정보/저장소

최고 점수만 localStorage에 저장합니다.
네트워크 통신, 쿠키 및 외부 추적 없음.
