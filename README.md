# 🐍 스네이크 게임 (HTML/CSS/JS 단일 파일)

키보드 **화살표 키(↑ ↓ ← →)** 로 조작하는 스네이크 게임입니다.  
별도 설치 없이 **index.html 하나만**으로 실행됩니다.  
**래핑 모드**(벽 통과), **속도 조절**, **최고 점수 저장(localStorage)** 기능을 제공합니다.

---

## ✨ 주요 기능
- HTML + CSS + JS **단일 파일**
- 키보드 화살표 키로 조작
- Space로 일시정지/재개
- **난이도(속도)** 선택: 느림 / 보통 / 빠름
- **래핑 모드**: 벽에서 반대편으로 이동
- 점수 & 최고 점수(브라우저 localStorage 저장)
- 반응형 UI
